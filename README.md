# 🚀 How To Run

## 1️⃣ Run Web App

### Option A

Open directly:

```bash
index.html
```

### Option B

Run local server:

```bash
python -m http.server 8000
```

Then open:

```bash
http://localhost:8000
```

---

## 2️⃣ Run Python CLI

### Scan clean resume

```bash
python resume_analyzer.py sampleresume.txt
```

### Scan flagged resume

```bash
python resume_analyzer.py flagged.txt
```

### Scan custom resume

```bash
python resume_analyzer.py your_resume.txt
```
