# ✨ Project Overview

Resume Shield AI is a futuristic resume analysis platform built using **HTML, CSS, JavaScript, and Python**.

The project helps users:
- detect risky or sensitive resume content
- improve ATS compatibility
- analyze resume vocabulary
- inspect flagged lines interactively
- upload PDF/TXT resumes
- customize detection rules dynamically

The project includes:
- 🌐 Interactive Web Dashboard
- 🐍 Python CLI Scanner
- 📄 PDF Support
- ⚡ Real-time Analytics
- 🛡️ Safety Detection Engine
- 📊 ATS Scoring System

---

# 🔥 Features

## 🛡️ Smart Resume Safety Scanner

Detects sensitive resume content related to:
- Substance abuse
- Suicide / self-harm
- Depression / emotional crisis
- Grief & traumatic personal loss

---

## 📊 ATS Structural Analysis

Checks whether the resume contains important ATS sections such as:
- Contact Information
- Career Objective
- Education
- Technical Skills
- Projects
- Experience
- Certifications

---

## 🖥️ Interactive Resume Inspector

- Highlights flagged lines
- Line-by-line resume inspection
- Detailed warning tooltips
- Category-wise flag breakdown
- Real-time document analysis

---

## ⚙️ Dynamic Rule Engine

Users can:
- Add custom categories
- Add/remove keywords
- Configure exclusions
- Save rules automatically using browser localStorage

---

## 📂 PDF & TXT Upload Support

Supports:
- `.txt`
- `.pdf`

Uses **PDF.js** for browser-side PDF parsing.

---

## 🎯 Vocabulary Strength Analysis

Detects professional action verbs such as:
- developed
- implemented
- optimized
- engineered
- created
- designed
- improved
- managed

# 🚀 How To Run The Project

# 1️⃣ Run The Web App

## ✅ Option A — Open Directly

Simply open:

```bash
index.html
```

---

## ✅ Option B — Run Using Local Server

Open terminal in the project folder and run:

```bash
python -m http.server 8000
```

Now open:

```bash
http://localhost:8000
```

---

# 2️⃣ Run The Python CLI Scanner

## ✅ Scan Clean Resume

```bash
python resume_analyzer.py sampleresume.txt
```

---

## ⚠️ Scan Flagged Resume

```bash
python resume_analyzer.py flagged.txt
```

---

## 📄 Scan Custom Resume

```bash
python resume_analyzer.py your_resume.txt
```

# ⚡ How It Works

1. Upload or paste resume text
2. Click **Analyze Resume**
3. The system scans the document
4. Dashboard updates in real-time
5. Flagged lines appear in inspector panel
6. ATS & vocabulary scores are calculated
7. Users can customize rules dynamically
